第一篇 如何搭建博客

一.下载nodeJS 并测试

1.地址https://nodejs.org/en/
2.测试node -v

二.安装hexo

1.npm install hexo -g

1.1 root Artasdlkajs

1.2 sudo npm install hero -g输入密码

2.测试 hexo

3.创建目录并进入

3.1 hexo init 初始化hexo目录

3.2 hexo server 启动本地服务

三.github账号

1.配置ssh-key

1.1 ssh-keygen 连续三次回车

1.2 vim~/.ssh/id_rsa.pub 复制ssh-key :q返回

1.3 github设置 ssh粘贴

2.配置github与本地机器验证

2.1 git config –global user.name””

2.2 gif config –global user.email””

2.3 新建博客域名仓库 用户名.github.io

四.config.yml

1.deploy字段

1.1 type:git

1.2 repo:域名仓库的https地址

五.发布

1.npm install hexo-deployer-git –save

2.hexo g

3.hexo d
