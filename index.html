H5新增标签

##HTML5新增标签

header定义文档的头部

footer定义文档尾部

section有语义的div作用与相关内容聚合

article更有语义的

aside定义侧边栏

nav定义导航

hgroup定义标题组合

details配合summary使用 summary应为detauls的第一个子元素属性open=”open”


dialog定义对话框open属性

figure定义图标figcaption定义图标标题

address定义文档作者信息通常包含在footer元素的其他信息中

短语句

mark 标记 高亮文本 常用语搜索高亮

meter定义度量衡 表示在一定范围内的值，如投票占比 使用量等

属性

value实际度量的值默认为0 可以为整数或者浮点数

high范围上限值low范围下限值

max最大值默认为1min最小值默认为0

optimum最佳值 必须在min与max之间

progress定义进度 属性value max min

wbr定义换行

abbr定义缩写The WHO was founded in 1948.

bdi定义隔离

cite blockquote q定义引入

cite行级斜体 blockquote块级整体带margin q块级带双引号

output 输出

pre code 格式化代码code单行 pre多行

##表单元素 input

tel用于输入电话号码

search 用于提示用户输入要搜索的文字

url用于输入单个网址

email 用于输入单个电子邮件地址如果指定了multiple属性就可以输入多个地址逗号分隔

datetime用于输入时区设置为UTC的日期和时间

date 用于输入不含时区的日期

month 用于输入含年份和月份但不含时区的日期

week 用于输入含年份和周数但不含时区的日期

time 用于输入含小时分钟秒钟和秒的小数部分但不含时区的时间值

datetime-local 用于输入不含时区的日期和时间

number 输入数字

range 输入数字不许输入具体数字 范围控件实施形式为滑块

color 用于通过颜色池控件选择颜色

lable 关联作用






datalist 配合list使用

ins定义被插入文档中的文本 del删除的文本

samp 定义计算机程序的样本文本

fieldset legend


标题

##表单属性

autofocus 用于网页加载后对焦到相关元素上的输入

placeholder 用于提示用户应输入的数据类型 浅色文字显示占位

required 提示用户应该输入字段

pattern 限制输入内容的格式内容

autocomplete 记忆之前输入的内容 on off

宽高单位

em

相对单位 继承父级字号设置

例：font-size:2em; 为父级字体大小的二倍

rem

相对单位 继承根（html）字号设置

例：font-size:2em; 为html字体大小的二倍

vw

窗口分为100份

例：width:20vw; 为窗口宽度百分之20

vh

窗口分为100份

例：height:20vh; 为窗口高度百分之20

maxwidth

最大宽度

maxheight

最大高度

minwidth

最小宽度

minheight

最小高度
